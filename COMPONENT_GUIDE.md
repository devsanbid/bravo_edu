- Card backgrounds

- Text (#1F2937 dark, #6B7280 light)
- Borders (#E5E7EB)
- Backgrounds (#F9FAFB)

```css
/* Primary Button */
from-primary-purple to-primary-purple-light

/* CTA Button */
from-accent-orange to-accent-gold

/* Decorative */
from-primary-purple via-purple-light to-accent-orange
```
